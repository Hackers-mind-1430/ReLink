# ReLink

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

ReLink2 is an Android application designed to help users reconnect with lost friends and discover new people based on shared interests. The app provides a modern and intuitive interface for building meaningful connections.

## 🌟 Features

* User authentication with email and password
* Create and edit a user profile (name, bio, profile picture, interests)
* Select multiple interests from 20+ categories
* Discover and reconnect with users with shared interests
* Real-time chat with friends
* Notifications for friend requests and messages
* Clean, responsive, and intuitive UI




## 🚀 Getting Started

Follow these instructions to get a local copy of the project up and running.

### Prerequisites

* Android Studio (Arctic Fox or higher recommended)
* Java 8+ or Kotlin support
* Firebase project configured with Authentication and Firestore



## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/hackers-mind-1430/ReLink.git
```

2. Open the project in Android Studio.
3. Connect your Firebase project:

   * Add `google-services.json` to the `app/` directory
   * Enable Firebase Authentication (Email/Password) and Firestore
4. Build and run the app on an emulator or physical device.


## 🎯 Usage

* Sign up with an email and password.
* Complete your profile by adding your name, bio, profile picture, and selecting interests.
* Browse and connect with users who share your interests.
* Chat in real-time and receive notifications for new messages or friend requests.
